# Kelab Robotik KV Sepang (HTML + CSS)

Kelab Robotik
- Daftar sebagai Ahli
- Senarai Ahli Sedia Ada

# Cara Guna

1. Laman Utama untuk meninjau perkara berkaitan Kelab Robotik KV Sepang.
2. Pergi ke halaman Daftar untuk mendaftar sebagai ahli.
3. Isi borang yang disediakan.
4. Tekan HANTAR untuk proses pendaftaran dijalankan.
5. Tekan RESET untuk mengosongkan ruang borang dan isi semula.
